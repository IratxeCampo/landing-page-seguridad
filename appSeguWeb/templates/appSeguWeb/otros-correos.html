{% trans 'Asunto' %}

{% blocktrans %} En este caso, el asunto del correo es especialmente llamativo y incita a la atención. {% blocktrans %} Si el asunto es vistoso y/o no se corresponde con el formato usual del resto de correos corporativos/universitarios, ándate con ojo. Obviamente, si ya el propio asunto presenta faltas de ortografía, esto podría ser también suficiente motivo de sospecha. {% endblocktrans %}

{% trans 'Remitente' %}

{% blocktrans %} Otro componente fundamental de todo correo electrónico dirigido a estudiantes, grupos de trabajadores, etc., es el remitente. Es aconsejable echar un vistazo al remitente del correo para asegurarse de que es una dirección de correo fiable, común con otros correos anteriores y que lleva el dominio correcto correspondiente a la entidad que se supone representa (no como el caso de nuestro correo, vuelve y mira si no te has fijado...). {% endblocktrans %}

{% trans 'Ortografía' %}

{% blocktrans %} En función del idioma utilizado, las faltas ortográficas y gramaticales son claros indicadores de falta de esmero y cuidado a la hora de confeccionar el correo, algo que, por lo general, es común en todo correo corporativo/universitario/otro. {% trans 'Otros' %}

{% blocktrans %} Otros puntos a destacar sobre este tipo de correos pueden ser la estructura general del correo (si está bien maquetado o no), si las imágenes tienen muy baja resolución, si son un solo bloque de texto sin mayor explicación y link a una página, si la propia organización o entidad ha avisado antes de otros ataques que se han dado con los estudiantes/trabajadores en el punto de mira… Por lo general, las campañas de ataque de phishing no suelen tener correos especialmente sofisticados preparados, ya que a lo mejor se utiliza una plantilla común para atacar a varias entidades, en distintos idiomas, cada una con sus particularidades, links, redes sociales, etc. {% endblocktrans %}
